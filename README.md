# Football Ticket Booking System

A relational database project built with PostgreSQL, covering schema design, ERD modeling, and SQL querying for a football ticket booking platform.

## Tech Stack
- PostgreSQL
- SQL

## Features
- Database schema with 3 normalized tables: Users, Matches, Bookings
- ERD with primary keys, foreign keys, and crow's foot notation
- 7 SQL queries covering filtering, pattern matching, NULL handling, JOINs, subqueries, and pagination

## Queries Covered
1. Filter by category and status
2. Pattern matching with LIKE and ILIKE
3. NULL handling with COALESCE
4. INNER JOIN across multiple tables
5. LEFT JOIN to include unmatched records
6. Subquery with AVG aggregation
7. Pagination with LIMIT and OFFSET
